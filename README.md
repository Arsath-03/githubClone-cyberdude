# Github Profile Clone
This project is a UI clone of a GitHub user profile page, created as part of my internship assignment. The main objective was to replicate the layout, design, and responsiveness of a real GitHub profile using frontend technologies.
# Tech Stack
- HTML
- Tailwind CSS
# Features
- GitHub-style profile layout
- Responsive design for mobile and desktop
- Mobile-friendly header navigation
- Profile sidebar and content sections
